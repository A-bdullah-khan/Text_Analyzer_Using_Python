# Text_Analyzer_Using_Python
Python based text analyzer using string operations
FEATURES:
1) Takes a text input from user
2) Ask for three letters and display their count
3) Converts letter into lowercase for counting
4) Calculate total number of letters in the text
5) Display the first and last letter of the text
6) Reverses the entire text
7) Checks whether the word PYTHON is present in the text or not
